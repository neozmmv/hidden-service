# Hidden Service

Simple FastAPI for exposing a Tor hidden service.

```bash
git clone https://github.com/neozmmv/hidden-service
```

```bash
python -m venv env
```

```bash
./env/Scripts/activate
```

```bash
docker compose up -d
```

```bash
pip install -r requirements.txt
```

```bash
fastapi dev main.py
```
